Get started with Amazon EC2

Use this tutorial to get started with Amazon Elastic Compute Cloud (Amazon EC2). You'll learn how to launch and connect to an EC2 instance. An instance is a virtual server in the AWS Cloud. With Amazon EC2, you can set up and configure the operating system and applications that run on your instance.

Overview

  • An image ‚Äì A template containing the software to run on your instance, such as the OS.
  • A key pair ‚Äì Credentials to prove your identity when connecting (public key on instance, private key on your computer).
  • A network ‚Äì A default Virtual Private Cloud (VPC) in every AWS Region for your account.
  • A security group ‚Äì Acts as a virtual firewall to control inbound/outbound traffic.
  • An EBS volume ‚Äì A root volume is required; data volumes are optional.

Tasks

Step 1: Launch an instance

  1. Open the Amazon EC2 console: https://console.aws.amazon.com/ec2/
  2. Optionally select AWS Region (top menu bar).
  3. In Launch instance pane, choose Launch instance.
  4. Give your instance a descriptive Name under Name and tags.
  5. Choose Quick Start for Application and OS Images (AMI), such as Amazon Linux (preferably Free Tier eligible).
  6. Select an Instance type marked Free Tier eligible.
  7. Choose or create a Key pair (login). (Required to connect via SSH or RDP.)
  8. Network settings:
    • Default VPC and subnet are preselected.
    • Security group allows connections from anywhere (0.0.0.0/0). Warning: Use Custom for production for better security.
  9. Configure storage (root volume preset, no data volumes required for tutorial).
  10. Review and Launch instance.
  11. Monitor status via notification and Instances page (pending ‚Üí running).

Step 2: Connect to your instance

The connection method depends on the instance OS.

  • Linux: Use SSH with your key pair.
  • Windows: Use RDP with decrypted Administrator password from your key pair.
If you have trouble connecting, see  Troubleshoot issues connecting to your Amazon EC2 instance.

Step 3: Clean up your instance

Warning: Terminating an instance is permanent and irreversible. Back up data as needed.

  1. In the navigation pane, choose Instances and select your instance.
  2. Choose Instance state > Terminate (delete) instance.
  3. Confirm Terminate (delete) in prompt.
  4. EC2 shuts down and terminates your instance. Entry disappears after a while.
